# TicTacToe

Local two player Tic Tac Toe game with a GUI

Made with JavaFX

HOW TO RUN THE GAME:

Launch the TicTacToeFX.jar file located in /TicTacToeFX/out/artifacts/TicTacToeFX_jar/
